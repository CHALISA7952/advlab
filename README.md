# advlab - YouTube Downloader

โปรแกรมดาวน์โหลดวิดีโอจาก YouTube ด้วยภาษา Python  
จัดทำเพื่อการศึกษาในรายวิชา Advanced Lab

## ไฟล์ในโปรเจกต์
- dlyoutube.py : โปรแกรมสำหรับดาวน์โหลดวิดีโอจาก YouTube

## หลักการทำงาน
1. ผู้ใช้ป้อนลิงก์ YouTube
2. โปรแกรมใช้ไลบรารี yt-dlp ในการดาวน์โหลดวิดีโอ
3. วิดีโอจะถูกบันทึกด้วยชื่อเดียวกับชื่อวิดีโอบน YouTube

## วิธีใช้งาน
```bash
python dlyoutube.py
